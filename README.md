Compiling
---------

###Linux:
To compile this project you need get a working **Code::Blocks IDE** and several libraries at least with these versions:

* libsdl-1.2.15
* sdl-net-1.2.8
* DevIL(OpenIL)-1.7.8
* CEGUI-0.7.7
* (optional) sdl-mixer-1.2.12
* (optional) OpenAL-1.15.1
* (optional) BulletPhysics-2.80
